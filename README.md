# 🌀 Pyriodic
Pyriodic is an open-source Python package for the analysis, visualisation and statistics of circular data build for respiration and other periodic signals. It includes modules for preprocessing, extraction of phase angles, visualisation, statistical analysis and more. 

![alt text](https://img.shields.io/pypi/dm/python-pyriodic.svg?label=PyPI)


## Documentation
[Documentation](https://laurabpaulsen.github.io/pyriodic/) for Pyriodic includes installation instructions, tutorials and API reference.

## Installation
To install the latest release of Pyriodic, run the following command:

```bash
$ pip install python-pyriodic
```
