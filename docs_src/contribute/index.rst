Contribute
==========

Thank you for considering contributing to ``Pyriodic``! It is an open-source
project maintained by `Laura Bock Paulsen <https://github.com/laurabpaulsen>`_.

There are many ways to contribute:

* Report bugs or request features by opening an issue on the
  `GitHub repository <https://github.com/laurabpaulsen/pyriodic/issues>`_.
* Contribute code or improve documentation by forking the dev branch of the repository, making your changes, and submitting a pull request.
* Improve existing tutorials or add new ones by submitting a pull request with your changes.
* Other contributions are welcome as well!
