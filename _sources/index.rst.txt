Welcome to Pyriodic's documentation!
=====================================

**Pyriodic** is a Python package for analysis of circular data. It was developed to facilitate analysis of respiratory data, but can be used for other types of circular data as well.

Check out the tutorials to get started, or jump to the API reference.
